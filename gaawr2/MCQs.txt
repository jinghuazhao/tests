Q2. Components of pipelines

Which of the following best illustrates the role of R in a Bash-based bioinformatics pipeline?

A. R replaces all command-line tools and directly performs sequence alignment.

B. R acts as one stage in a workflow, receiving data from upstream tools and producing outputs that can be passed to downstream tools.

C. R is used only for plotting final results and cannot be integrated into automated pipelines.

D. R can only read data from interactive user input.

Answer: B. R acts as one stage in a workflow, receiving data from upstream tools and producing outputs that can be passed to downstream tools.

Q2. In the diabetes analysis pipeline, which sequence best represents the workflow used?

A. group_by → mutate → filter → summarize
B. filter → mutate → group_by → select → summarize
C. summarize → filter → mutate → group_by
D. select → summarize → filter → mutate

Answer: B. filter → mutate → group_by → select → summarize

Explanation: The example pipeline filters records, recodes variables with mutate(), groups data, selects variables, and finally summarizes them.
